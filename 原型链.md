  所有引用类型（函数，数组，对象）都拥有__proto__属性（隐式原型）;
  ---
  所有函数拥有prototype属性（显式原型）（仅限函数）
  ---


-------------------------------------------------------------------
__proto__是每个对象都有的一个属性，而prototype是函数才会有的属性!!!

```
function Fn() { }
        Fn.prototype.name = '王福朋';
        Fn.prototype.getYear = function () {
            return 1988;
        };
```
